AWeber Control Panel

Design update for the AWeber control panel

Overview

Role: Discovery, UI Design, HTML/CSS

During our recent refresh of the AWeber brand, I identified an opportunity to update the control panel. After fleshing out the opportunity to the design team, I was given the green-light to lead the redesign and we produced a control panel experience that was on-brand, intuitive, and highly functional.

Challenge

Our design team didn't have a current style guide so we had to start from scratch. Working with another designer, we spent the first few weeks mapping out our entire control panel experience. We identified the components and shared elements in each area and started defining patterns. My main goal from the redesign was to not only define a basic style guide we could work with but to also create a responsive experience starting mobile-first, which was a major overhaul for our current application.

Solution

In a little under 2 months, we were able to update and define all of the UI elements including fonts, component designs, and colors. In addition to the design and code, we delivered a UI kit which formed the basis for the global AWeber style guide
